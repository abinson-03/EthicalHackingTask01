# Ethical Hacking Reconnaissance Task

This repository contains the results of a passive information gathering and reconnaissance exercise for an Ethical Hacking assignment.

## Overview
The goal of this task was to perform passive reconnaissance on a target domain (`mit.edu`) without exploiting or accessing any unauthorized systems. 

### Key Findings
- **WHOIS Information**: Domain registration details and name servers.
- **DNS Enumeration**: A, MX, NS, and TXT records.
- **Technology Stack**: Web server identification and CDN usage.
- **Security Headers**: Analysis of HTTP security policies (CSP, HSTS, X-Frame-Options).
- **Robots & Sitemap**: Insights derived from `robots.txt` configuration and sitemap availability.

## Files
- `Reconnaissance_Report.md`: The complete, structured reconnaissance report.
- `whois.txt`, `dns_*.txt`, `headers.txt`, etc.: Raw data outputs from the various reconnaissance commands.

*Disclaimer: All reconnaissance was strictly passive and strictly adhered to ethical hacking guidelines, utilizing only publicly available information.*
